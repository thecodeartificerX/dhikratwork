# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | Yes                |

## Reporting a Vulnerability

If you discover a security vulnerability in DhikrAtWork, please report it responsibly:

1. **Do not open a public issue.** Security vulnerabilities should not be disclosed publicly until a fix is available.

2. **Use GitHub Security Advisories.** Go to the [Security Advisories](https://github.com/thecodeartificerX/dhikratwork/security/advisories) page and click "Report a vulnerability."

3. **What to include:**
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

4. **Response timeline:**
   - Acknowledgment within 48 hours
   - Status update within 7 days
   - Fix released as soon as practical

Thank you for helping keep DhikrAtWork safe.
